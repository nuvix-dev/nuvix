**Welcome to Nuvix!**

Nuvix is a powerful Managed Backend-as-a-Service (BaaS) platform built with Fastify and NestJS. It provides scalable, real-time APIs for managing users, authentication, databases, storage, messaging, and more. Nuvix is designed to give developers full control over their backend infrastructure.

### Features

- **Fastify and NestJS Based APIs**
- **Authentication & User Management**
- **Database Management (SQL)**
- **File Storage with Buckets**
- **Real-time APIs with WebSockets**
- **GraphQL Support**
- **Messaging & Notifications**
- **Multi-Tenant Support**
- **Role-Based Access Control (RBAC)**

### Quick Start

1. **Clone the Repository**:

```bash
git clone https://github.com/Nuvix-Tech/nuvix.git
cd nuvix
```

2. **Install Dependencies**:

```bash
npm install
```

3. **Run the Server**:

```bash
npm run start
```

4. **Development Mode**:

```bash
npm run start:dev
```

5. **Production Mode**:

```bash
npm run start:prod
```

### System Requirements

- Node.js >=18.0.0
- NPM >=9.0.0
